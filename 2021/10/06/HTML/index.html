# 3. 图像标签

alt 标签：图片无法正常显示时，显示文字描述

<img src="../resources/image/1.jpg" alt="头像" title="鼠标悬停时显示的文字">


<img src="../resources/image/1.jpg" alt="头像" title="鼠标悬停时显示的文字" width="300" height="300">

还可以设置图片宽和高等等…

# 4. 链接标签
  • 文本超链接

  • 图像超链接

将图片标签嵌入链接标签中

  • 锚链接

多应用于页面的跳转，如目录

  • 功能性链接
<!--功能性链接 邮件链接 mailto:+邮箱地址   -->
<a href="mailto:2304465670@qq.com">点击联系我</a>

可以打开邮箱

# 5. 行内标签和块元素
  • 块元素

无论内容多少，该元素独占一行 如：p、h1-h6…

  • 行内元素

内容撑开宽度，左右都是行内元素的可以排在同一行 如：a、strong、em…
